# Sales Analytics System

A Python-based Sales Analytics System that reads sales data, performs analysis, enriches data using an external API, and generates a comprehensive sales report.

---

## Features

- Read and parse sales data from file
- Perform sales analytics:
  - Total revenue calculation
  - Region-wise sales
  - Top-selling products
  - Customer analysis
  - Daily sales trends
  - Peak sales day
  - Low-performing products
- Fetch product details from an external API
- Enrich sales data with API information
- Save enriched data to file
- Generate a detailed sales report

---
